# [Astro Records](https://astro-spa.netlify.app)
![image](https://github.com/RayyanNafees/astro-records/assets/59685062/396fe255-0559-4d58-a8da-ade5fc6f2004)
An astro Music Library harnessing the power of the new [Experimental ViewTransitions API](https://docs.astro.build/en/guides/view-transitions/)

## Tech Stack:
-  [Astro](https://astro.build)
-  [Tailwind](https://tailwindcss.com)
-  [Preact](https://preactjs.com)
-  [Cloudflare Workers](https://workers.cloudflare.com)
